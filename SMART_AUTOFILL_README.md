# 智能表单自动填写（学习型）

## 功能特点

- **学习型**：你填写一次，它就记住，下次自动填写
- **不侵入**：不会新增页面元素，只填写已有的空字段
- **可导出**：随时导出 JSON 配置备份
- **智能识别**：通过 name、id、placeholder、label 等识别字段

## 使用方法

### 安装脚本
将 `userscript/autofill-smart.user.js` 安装到 Tampermonkey

### 首次使用
1. 打开任意表单页面
2. 像往常一样手动填写表单
3. 脚本会自动学习你填写的内容

### 再次访问
1. 打开同类表单页面
2. 空字段会自动被填写（绿色高亮）
3. 或点击「自动填写当前页面」按钮

### 导出/导入配置
- 点击「导出配置」保存为 JSON 文件
- 点击「导入配置」加载之前的备份

## 快捷键

- `Alt + S`：显示/隐藏面板

## 工作原理

脚本通过以下方式识别字段：
1. `name` 属性
2. `id` 属性
3. `placeholder` 属性
4. 关联的 `label` 文本
5. `aria-label` 属性
6. 页面路径 + 类型 + 位置

## 注意事项

- 不会覆盖已有内容的字段
- 敏感信息（密码等）也会被记录，请注意安全
- 建议定期导出配置备份
